PRINCIPALES SETENCIAS DE MYSQL

MySQL comparte las sentencias del lenguaje SQL. Lo que en la practica se traduce en una plena
compatibilidad, algunas sentencias básicas son las siguientes:



  • SELECT es usada para consultar datos.
  • WHERE es usada para incluir condiciones en
    los datos que queremos consultar.
  • UPDATE es usada para actualizar o modificar datos ya existentes.
  • DELETE es usada para borrar datos de una tabla.
